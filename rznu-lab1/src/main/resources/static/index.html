Dokumentacija Tweeter REST API-ja

"/users" -> svi korisnici
- dostupne HTTP metode: GET (dohvat svih korisnika), POST (dodavanje korisnika)

"/users/{user_id}" -> korisnik s identifikatorom user_id
- dostupne HTTP metode: GET (dohvat korisnika), PUT (ažuriranje korisnika), DELETE (brisanje korisnika)

"/tweets" -> svi tweetovi
- dosuptne HTTP metode: GET (dohvat svih tweetova), POST (dodavanje tweeta)

"/tweets/{tweet_id}" -> tweet s identifikatorom tweet_id
- dosuptne HTTP metode: GET (dohvat tweeta), PUT (ažuriranje tweeta), DELETE (brisanje tweeta)

"/users/{user_id}/tweets" -> tweetovi korisnika s identifikatorom user_id
- dosuptne HTTP metode: GET (dohvat tweetova)
